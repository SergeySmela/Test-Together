Правила Игры

  • Цель игры - за 1 минуту набрать наибольшее количество очков убирая кубики с поля
  • Кубики убираются с поля наведением курсора на кубик и последующим кликом левой кнопкой мышки
  • Для начала Игры нажмите New Game
  • Для остановки/запуска таймера нажмите Start
